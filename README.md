# 🚀 Iron Dome

A browser-based missile defense game inspired by Angry Birds and the Israeli Iron Dome air defense system.

**[▶ Play it here](https://r94r.github.io/irondomegame/)**

---

## How to Play

Intercept incoming missiles before they destroy your city.

### Controls
- **Hold & drag** anywhere → pull back the slingshot to aim
- **Release** → launch the interceptor
- **Keys `1` / `2` / `3`** → switch interceptor type
- **`Esc` or `P`** → pause

### Interceptor Systems

| System | Key | Target | Ammo |
|---|---|---|---|
| Iron Dome | `1` | Small missiles | ∞ |
| David's Sling | `2` | Medium missiles | Limited |
| Arrow | `3` | Large & medium missiles | Limited |

> ⚠️ Using the wrong interceptor type won't work — match the right system to the right threat!

### Waves
- **5 waves**, each more intense than the last
- A **3-2-1-GO** countdown starts each wave
- Ammo is replenished between waves
- **Lose** if all buildings are destroyed
- **Win** if you survive all 5 waves

---

## Features
- Slingshot aiming mechanic (Angry Birds style)
- 3 missile sizes with realistic missile visuals & smoke trails
- Detailed Iron Dome battery launcher with radar dish & blinking light
- Destructible city with partial damage, crumbling buildings & rubble physics
- Explosion particles & shockwaves
- Synthesized sound effects (Web Audio API — no audio files needed)
- Mario Kart-style 3-2-1-GO countdown beeps
- Mobile friendly (touch support)
- Pause button + keyboard shortcut

---

## Run Locally

Just open `index.html` in any modern browser — no server or build step needed.
